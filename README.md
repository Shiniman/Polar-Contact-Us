## Getting Started

### Prerequisites
- Node.js
- npm (Node Package Manager)
- Git

### Installation

1. Clone the repository:
    ```sh
    git clone https://github.com/pol-ar/company-site.git
    ```

2. Navigate to the project directory:
    ```sh
    cd company-site
    ```

3. Install the dependencies:
    ```sh
    npm install
    ```

4. Run the development server:
    ```sh
    npm start
    ```
